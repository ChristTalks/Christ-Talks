# Pathway - Expo Starter (TikTok-style feed)

## Run locally
1. Install Node.js (>=16).
2. Install Expo CLI (optional): `npm install -g expo-cli`
3. Install dependencies: `npm install`
4. Start dev server: `npm run start`
5. Use Expo Go on your phone to scan the QR code, or run in simulators.

## Run on Replit
1. Create a new Repl, choose "Import from GitHub" or create a new "Node" or "Expo" Repl.
2. Paste project files into the Repl file tree.
3. Run `npm install` in the Repl shell.
4. Run `npm start` and follow Expo web preview or use the link/QR.
